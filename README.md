# Agentic Research App

## To run

Create a `.env.compose` file with

```
SERPAPI_API_KEY=...
OPENAI_API_KEY=...
GEMINI_API_KEY=...
```

then

```
docker-compose up
```

## To test
 
 1) create your user (namespace)
   - add username
   - select a personality
   - submit
 2) select a document (yours or one from the resources folder) and hit Analyze
 3) ask questions about the doccument
